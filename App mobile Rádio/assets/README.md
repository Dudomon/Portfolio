# Assets do App

## Estrutura de Pastas

```
assets/
├── fonts/          # Fontes Montserrat
├── images/         # Imagens e backgrounds
└── icons/          # Ícones do app
```

## Fontes Necessárias

Baixe as fontes Montserrat em: https://fonts.google.com/specimen/Montserrat

Arquivos necessários:
- `Montserrat-Regular.ttf`
- `Montserrat-Medium.ttf`
- `Montserrat-SemiBold.ttf`
- `Montserrat-Bold.ttf`

## Imagens Recomendadas

- `logo.png` - Logo da rádio (512x512)
- `background.jpg` - Background opcional
- `placeholder.png` - Placeholder para imagens

## Ícones

- Ícone do app em várias resoluções
- Ícones customizados se necessário
